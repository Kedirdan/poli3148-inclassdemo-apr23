## Running from scratch

```bash
# 1. Install dependencies
pip3 install -r requirements.txt

# 2. Generate the dashboard (uses the committed data/processed/vdem_europe.csv)
python3 dashboard.py
#  → writes dashboard.html (root) AND docs/index.html (GitHub Pages)
```

Open `dashboard.html` directly in any modern browser.

### Obtaining the raw data

The 32 MB `data/raw/vdem-v16.RData` is gitignored. To regenerate the processed
CSV from scratch:

```bash
# Fetch V-Dem v16 from the official R-package release
mkdir -p data/raw
curl -L https://api.github.com/repos/vdeminstitute/vdemdata/tarball/V16 \
  | tar xz -C data/raw --strip-components=1 --wildcards '*/data/vdem.RData'
mv data/raw/data/vdem.RData data/raw/vdem-v16.RData
rmdir data/raw/data

# Then re-extract the Europe slice
pip3 install pyreadr
python3 data_prep.py
```

## GitHub Pages deployment

This project is set up to deploy to GitHub Pages using the `/docs` folder:

1. Push this repository to GitHub.
2. In the repository settings → **Pages**, set:
   - **Source**: Deploy from a branch
   - **Branch**: `main` / folder: `/docs`
3. After ~30 seconds the dashboard is live at
   `https://<username>.github.io/POLI3148_2026Spring_GitHubPagesDemo/`.

The `docs/index.html` file is automatically regenerated by `dashboard.py` — keep
both the root `dashboard.html` (for course submission) and the `docs/` copy
(for the live page) in sync by running `python3 dashboard.py` after any change.

  `v2x_regime` (Regimes of the World classification, 0–3).

## The five sections

1. **Rise and Fall (1900–1945).** Europe's interwar collapse — from polyarchy peak
   0.595 in 1923 to trough 0.118 in 1942, with Latvia (-0.609), Germany (-0.575),
   and Lithuania (-0.463) the sharpest fallers.
2. **A Divided Continent (1945–1990).** The 40-year Cold War bifurcation (West mean
   0.75, East mean 0.16), and the Iberian / Greek authoritarian interlude inside
   "Western" Europe.
3. **The Third Wave's European Chapter (1989–2010).** Post-communist trajectories
   diverge — Czechia's +0.737 gain, Poland's +0.513, contrasted with Russia's
   -0.192 and Belarus's -0.339.
4. **The Backsliding Wave (2010–2024).** Liberal-democracy decline across 30+
   European states — Hungary (-0.352), Serbia (-0.278), Türkiye (-0.251),
   Greece (-0.247), Poland (-0.214).
5. **Europe 2024 — A Regime Typology.** V-Dem's 4-way RoW classification of 42
   European states: 19 liberal democracies, 17 electoral democracies, 5
   electoral autocracies, 1 closed autocracy (Belarus).

## Design choices

- **Single self-contained HTML.** Plotly.js is loaded from CDN; the file is
  ~107 KB on disk and opens offline after a one-time CDN fetch.
- **Cold-war-archive aesthetic.** Aged parchment (`#F3EAD8`) background, deep
  oxblood (`#7A2E2E`) and navy (`#1E3A5F`) accents, Playfair Display serif
  headers, IBM Plex Mono for metadata — chosen to evoke printed political
  yearbooks and reflect the 20th-century subject matter.
- **Analytical text is hand-authored.** Every number in the prose comes from
  `explore.py` output; no "globally, democracy has declined" generic claims.
